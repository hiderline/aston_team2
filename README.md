## Обзор
Консольное приложение для управления и сортировки коллекции объектов класса Автобус (Bus).
Проект демонстрирует применение:

- Основных принципов ООП
- Паттернов проектирования
- Алгоритмов сортировки
- Способностей разработки на Java


## Особенности
**Основной функционал**
- Множественные способы заполнения: ручной ввод, случайная генерация, загрузка из файла
- Сортировка по всем полям: номер, модель, пробег
- Комбинированная сортировка: по нескольким полям одновременно
- Паттерн "Стратегия": гибкая система алгоритмов сортировки
- Встроенная валидация: проверка всех входных данных
- Интерактивное меню: циклическое выполнение с интуитивным интерфейсом
- Паттерн Builder: безопасное создание объектов Bus

**Дополнительный функционал**
- Кастомная сортировка: объекты с четными значениями сортируются, с нечетными остаются на местах
- Экспорт данных: автоматическое сохранение результатов в файл
- Stream API: современные методы обработки данных
- Многопоточность: параллельный поиск в коллекциях

## Архитектура
| **Паттерн** | **Назначение**                           | **Реализация**            |
|-------------|------------------------------------------|---------------------------|
| Strategy    | Динамическая смена алгоритмов сортировки | SortStrategy и реализации |
| Builder     | Пошаговое создание сложных объектов      | Bus.Builder               |
| Facade      | Упрощённый интерфейс к сложной системе   | ApplicationFacade         |

## Установка и запуск
### Клонирование репозитория
```bash
git clone https://github.com/hiderline/aston_team2.git
cd bus-sorting-app
```
### Сборка проекта
```bash
mvn clean compile
```
### Запуск приложения
#### 1. Через Maven
```bash
mvn exec:java -Dexec.mainClass="teamwork.Main"
```
#### 2. Через JAR файл
```bash
mvn clean package
java -jar target/bus-sorting-app.jar
```
#### 2. Через JAR файл
```bash

```
#### 3. Через IDE
```
1. Откройте проект в вашей IDE
2. Найдите класс Main.java
3. Запустите метод main()
```

## Пример использования
### Главное меню
```text
=== Главное меню ===
1. Заполнить коллекцию
2. Показать коллекцию
3. Отсортировать коллекцию
4. Сохранить в файл
5. Найти количество вхождений
6. Очистить коллекцию
0. Выход
```
### Заполнение коллекции
```text
=== Заполнение коллекции ===
1. Вручную
2. Случайно
3. Из файла
0. Вернуться назад
```

```text
=== Сортировка ===
1. По номеру

3. По пробегу
4. По трём полям сразу
5. По чётному числовому полю
0. Вернуться назад
```
